Проект 1С загрузчиков
=====================

Схема разработки
----------------
Ветка develop - для разработки, можно заводить sub-feature-branches.
Ветка master - для релизов, отсюда забираются готовые версии для тестов и клиентов. Сюда мержить только проверенные версии ПО!

В этом репозитории хранятся только загрузчики для последней версии iiko.

Бинарные файлы с COM-объектом брать из текущей сборки РМС, с которой работает разработчик.

Разработка строго по issues - всегда заводить задачу и указывать ее номер в commit message.
