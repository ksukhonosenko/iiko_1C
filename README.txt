Проект 1С загрузчиков
=====================

Схема разработки
----------------

Разработка строго по issues - всегда заводить задачу и указывать ее номер в commit message.
Заводятся, например, отсюда - https://github.com/ksukhonosenko/iiko_1C/issues?state=open

Работа с ветками:
- ветка develop - для разработки, можно заводить sub-feature-branches.
- ветка master - для релизов, отсюда забираются готовые версии для тестов и клиентов. Сюда мержить только проверенные версии ПО!



Бинарные файлы с COM-объектом брать из текущей сборки РМС, с которой работает разработчик.

Пока В этом репозитории хранятся только загрузчики для последней версии iiko.
Если мы будем дорабатывать КОМ-объект и добавлять новые функции, то загрузчики должны будут отражать эти изменения.
Тогда будут созданы папки - загрузчики для версии 3.ххх, загрузчики для версии 4.ххх и тп.
