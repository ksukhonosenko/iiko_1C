1. Откройте элемент меню Сервис->Дополнительные отчеты о обработки->Дополнительные внешние обработки
2. Нажмите "Добавить"
3. Выберите файл обработки iikoRMS.epf (По умолчанию из C:\Program Files\iikoRMS\Выгрузка в 1С\1С 8.1\iikoRMS.epf) 
4. Заполните поле название: "Загрузка из iikoRMS"
5. Нажмите "OK"
6. Используйте обработчик "Загрузка из iikoRMS" из элемента меню Сервис->Дополнительные отчеты о обработки->Дополнительные внешние обработки
